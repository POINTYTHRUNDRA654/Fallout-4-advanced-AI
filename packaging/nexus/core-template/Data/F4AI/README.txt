Welcome to Fallout 4 Advanced AI!

This mod is part of the Mossy Industries ecosystem, bringing
advanced AI-powered NPC dialogue to Fallout 4.

═══════════════════════════════════════════════════════

✨ FEATURES:

• True one-click installation (MO2 & Vortex)
• Fully offline - no cloud services or API keys
• Bundled Piper TTS for voice synthesis
• Real-time AI dialogue generation via KoboldCPP
• Conversation memory system
• Optional Mossy Industries integration

═══════════════════════════════════════════════════════

🚀 QUICK START:

1. Install via Mod Organizer 2 or Vortex
2. Download KoboldCPP and a GGUF model
3. Run Launch_F4AI_Bridge.bat
4. Start Fallout 4 with F4SE
5. Talk to NPCs and enjoy AI conversations!

See FIRST_RUN.txt for detailed instructions.

═══════════════════════════════════════════════════════

🔧 MOSSY INTEGRATION (OPTIONAL):

This mod works perfectly standalone, but can integrate
with the Mossy Industries platform for:

• Unified memory across games
