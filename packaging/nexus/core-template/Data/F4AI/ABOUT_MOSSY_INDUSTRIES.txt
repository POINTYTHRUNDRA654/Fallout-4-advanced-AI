**Fallout 4 Advanced AI** is a specialized implementation of Mossy Industries technology for Fallout 4, designed to work both standalone and as part of the larger Mossy ecosystem.

---

## Integration with Mossy

This mod can operate in two modes:

### Standalone Mode (Default)
- Uses local KoboldCPP for AI generation
- Uses bundled Piper TTS for voice synthesis
- Fully offline, no external services required
- Perfect for users who want a simple, self-contained solution

### Mossy Integration Mode (Optional)
- Connects to Mossy Industries endpoint
- Leverages Mossy's advanced AI capabilities
- Unified conversation memory across games
- Enhanced emotion detection and character consistency
- Enable by setting `enable_mossy_bridge: 1` in config.json

## Technical Architecture

### Standalone Architecture:
```
Fallout 4
	↓
F4AI_Core.esp (Papyrus scripts)
	↓
Fallout4_AI_Engine.exe (Python bridge)
	↓
KoboldCPP (local LLM) ← User provides
	+
Piper TTS (bundled) ← Included in mod
```

### Mossy Integration Architecture:
```
Fallout 4
	↓
F4AI_Core.esp (Papyrus scripts)
	↓
Fallout4_AI_Engine.exe (Python bridge)
	↓
Mossy Bridge (optional)
	↓
Mossy Core (centralized AI)
	↓
Multiple AI backends
	+
Unified memory system
	+
Cross-game features
```

## Configuration

Enable Mossy integration in `Data/F4AI/config.json`:

```json
{
  "enable_mossy_bridge": 1,
  "mossy_endpoint": "http://127.0.0.1:8765/f4ai/bridge",
  "mossy_timeout": 3.0
}
```

Default is `0` (disabled) for standalone operation.
