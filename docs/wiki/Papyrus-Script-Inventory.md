# Papyrus Script Inventory

Core script templates in this repository:

* `papyrus/F4AI_QueueManager.psc` — async dialogue queue and response playback controller.
* `papyrus/F4AI_CrowdNPC.psc` — per-NPC queue registration hook (project-specific integration).
* `papyrus/F4AI_VisionWidgetManager.psc` — vision/crosshair trigger output.
* `papyrus/F4AI_PushToTalkTrigger.psc` — microphone push-to-talk trigger and response wait loop.
* `papyrus/F4AI_InterNpcManager.psc` — inter-NPC multi-turn conversation trigger.
* `papyrus/F4AI_FeedbackMonitor.psc` — training feedback capture (+/- reward signals).
