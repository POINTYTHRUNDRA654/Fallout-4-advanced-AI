## Key Messages

### Primary Message:
"Advanced AI-powered NPC dialogue for Fallout 4, brought to you by Mossy Industries. Works standalone or as part of the Mossy ecosystem."

### Key Features:
- Part of the Mossy Industries AI platform
- True one-click installation
- Fully offline (no API keys)
- Optional Mossy integration for advanced features
- Professional-grade voice synthesis

### Target Audience:
- Fallout 4 players wanting immersive AI conversations
- Users interested in local AI without cloud services
- Modders wanting to extend AI functionality
- Mossy ecosystem users wanting cross-game AI

## Dos and Don'ts

### ✅ DO:
- Use "Mossy Industries" in full name
- Include subsidiary tagline
- Link to Mossy ecosystem when relevant
- Maintain professional tone
- Emphasize both standalone and integrated modes

### ❌ DON'T:
- Use "Mossy" alone without context
- Imply Mossy Industries is a commercial company
- Require Mossy integration for basic features
- Hide that Mossy is open source
- Use unofficial logos or branding
